# 🏰 Golgari Palace Game Day

Board game collection browser with photos from game nights.

## 🚀 Quick Start

**Requirements:**
- Node.js 20+ (upgrade from 18 first!)

**Run locally:**
```bash
npm install
npm start
```
Open http://localhost:4200

**Deploy to GitHub Pages:**
```bash
npm run deploy
```

Done! Your site is live on GitHub Pages.
