# Undercity Wilderness Expansion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the Wilderness into a distinct, dangerous central region — its own ashen-bramble look, a walkable causeway to the floating isle, grown to 32 nodes in a bigger world, and filled with a dedicated T2+ enemy pool.

**Architecture:** Backend adds two wilderness monster pools and makes `_wild_battle` region-aware so every wilderness fight (wild *and* elite) draws from them. `map.json` is expanded (world size + 18 new wilderness nodes incl. a 6-node causeway to `isl_warp`) via a verified generator script, then mirrored. The Angular board renderer gains an ashen-bramble `REGION_THEME` so the Wilderness stops using the cavern fallback.

**Tech Stack:** Python 3.11 Lambda (pytest, in-memory FakeTable suite), Angular 20 canvas renderer (TypeScript). Map source of truth `infrastructure/lambda/map.json`, mirrored to `public/data/undercity-map.json` via `sync_map.py`.

**Spec:** `specs/2026-07-20-undercity-wilderness-expansion-design.md`

**Run backend tests with:** `cd infrastructure/lambda && python -m pytest tests -q`

```python
def _wild_battle(table, sid, doc, elite=False, region=None):
    """Landing on a wild/elite space STARTS an interactive battle (Plan 2).
    In the 'wilderness' region both wild AND elite spaces pull from the tougher
    T2+ wilderness pools."""
    biome = data.dungeon_biome(doc.get('position', ''))
    if biome:
        spec = data.DUNGEON_NPCS[biome]          # dungeon fauna, themed per pocket
    elif region == 'wilderness':
        spec = _rng.choice(data.WILDERNESS_ELITE_NPCS if elite else data.WILDERNESS_NPCS)
    else:
        spec = _rng.choice(data.ELITE_NPCS if elite else data.NPCS)
    npc = engine.npc_from_spec(spec)
    npc['personality'] = spec.get('personality', data.NPC_DEFAULT_PERSONALITY)
    npc['bluff'] = spec.get('bluff', data.NPC_DEFAULT_BLUFF)
    return _start_battle(table, sid, doc, 'elite' if elite else 'wild', npc,
                         node=doc.get('position'))
```

```bash
 ('w_b1','elite',   1950,1560, ['w_m0','wild_gar0']),
 ('cw0','wild',     1540,1250, ['w_u1']),
 ('cw1','elite',    1560,1160, ['cw0']),
 ('cw2','hazard',   1550,1000, ['cw1']),
 ('cw3','wild',     1600,890,  ['cw2']),
 ('cw4','elite',    1690,890,  ['cw3']),
 ('cw5','wild',     1665,995,  ['cw4','isl_warp']),
]
assert not any(nid in byid for nid,*_ in new), 'wilderness expansion already applied'
for nid,typ,x,y,nb in new:
    nd={'id':nid,'type':typ,'x':x,'y':y,'region':'wilderness','neighbors':list(nb)}
    m['nodes'].append(nd); byid[nid]=nd
# auto-symmetrize every edge (adds reverse links onto existing/isle nodes too)
for n in m['nodes']:
    for nb in list(n['neighbors']):

            byid[nb]['neighbors'].append(n['id'])
m['regions']['wilderness']={'label':'The Ashen Wilds',
    'background':'undercity/arena_background.png','scatter':True,'dark':False}
json.dump(m,open(p,'w',encoding='utf-8'),indent=1,ensure_ascii=False)
print('total', len(m['nodes']), '| wilderness', sum(1 for n in m['nodes'] if n['region']=='wilderness'))
"
```
