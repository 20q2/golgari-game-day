# Squirrel Signature Spell (Acorn Fury) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give the Squirrel race a unique always-castable signature spell, Acorn Fury (+2 ATK next battle), via a new additive species-innate slot alongside the existing biome innate.

**Architecture:** Server is the source of truth. Add one `self_buff` spell and a `SPECIES_SPELLS` map keyed on the stored `species`; the innate-cast check accepts a spell that matches the biome OR species innate. `effective_stats` gains one `elif` (respecting the Squirrel Warrior `mult` doubling). The client mirror is regenerated by `sync_spells.py`; the Creature tab shows both innates in one group and the board cast picker offers both under the `innate` source.

**Tech Stack:** Python 3.11 Lambda (pure functions + pytest), Angular 20 standalone components, generated TS mirror.

Design spec: [2026-07-23-undercity-squirrel-signature-spell-design.md](2026-07-23-undercity-squirrel-signature-spell-design.md).

```typescript
/** Every always-castable innate spell id for a creature: biome innate plus any
 *  species signature (squirrels get Acorn Fury). Order: biome first. */
export function innateSpellIds(homeBiome: string | undefined, species: string | undefined): string[] {
  const ids: string[] = [];
  const biome = BIOME_SPELLS[homeBiome ?? ''];
  if (biome) ids.push(biome);
  const sig = SPECIES_SPELLS[species ?? ''];
  if (sig && !ids.includes(sig)) ids.push(sig);
  return ids;
}
```

## Notes for the implementer

- **Species field:** the player doc's `species` is the starter id (e.g. `'squirrel'`) and persists through evolution, so Squirrel Warrior/Mage/Archmage all keep Acorn Fury. Client `YouDoc` exposes `species`.
- **Warrior doubling is automatic:** the `self_buff` cast path already stamps `mult` on the buff entry for `spell_warrior` creatures; Task 2's `* mult` picks it up. No extra wiring.
- **Haste is automatic:** `spell_haste` already halves cooldowns in `_start_spell_cooldown` for any spell, so Acorn Fury's 15 min → 7.5 min for squirrels needs no new code.
- **Stacking is intended:** a garden squirrel can hold both `rot_surge` (+3) and `acorn_fury` (+2) → +5 ATK for one fight. Different `buffKind`s, so `_apply_buff` refresh-don't-stack does not merge them. No cap by design.
